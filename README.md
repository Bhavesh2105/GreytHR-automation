# GreytHR-automation

# Requirements:

- Set up a config folder
- inside config create a file *default.json*
 ## default.json
 ``` json
 {
  "username":"<Your Username>",
  "password":"<Your Password>"
 }
 ```
- To install the modules run
```
  npm install 
```
- to Sign in and sign out run 
```
node index.js
```
